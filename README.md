📊 Sales Data Analysis using SQL, Python & Power BI
📌 Project Overview

This project analyzes customer sales data to uncover business insights using SQL, Python, and Power BI. The goal is to clean data, perform exploratory analysis, calculate KPIs, and build an interactive dashboard for decision-making.

🎯 Objectives
Analyze customer purchasing behavior
Identify top-performing products and customers
Calculate sales KPIs
Build an interactive Power BI dashboard
Generate business insights using SQL and Python
